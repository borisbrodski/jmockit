Annotation Type Cascading

@Documented
@Retention(value=RUNTIME)
@Target(value={FIELD,PARAMETER})
public @interface Cascading

Indicates a mocked type where methods which return a reference type, excluding String, primitive wrappers, and collection types, will be automatically mocked if and when an invocation to the method occurs. Instead of returning the default null reference, such methods will return a mocked instance of the return type on which further invocations can be made. This behavior automatically cascades to those mocked return types.
