Package mockit.integration.logging Description

Provides integration with logging frameworks such as Log4j, SLF4j, and standard JDK logging, with the purpose of isolating code under test from them.

Contains one mock class for each supported logging framework, to be used with the UsingMocksAndStubs annotation when applied to a test class.
