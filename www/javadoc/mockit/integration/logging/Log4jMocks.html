Class Log4jMocks

java.lang.Object
  mockit.integration.logging.Log4jMocks

@MockClass(realClass=org.apache.log4j.Logger.class,
           instantiation=PerMockSetup,
           stubs={"trace","isTraceEnabled"})
public final class Log4jMocks
extends Object

A mock class containing mocks and stubs for the Log4j API.

When a test class is annotated as @UsingMocksAndStubs(Log4jMocks.class), all production code touched by the tests in that class will receive mock Logger instances instead of real ones, when one of the factory methods in class org.apache.log4j.Logger is called.
