Class Slf4jMocks

java.lang.Object

           stubs="getILoggerFactory")
public final class Slf4jMocks
extends Object

A mock class containing mocks and stubs for the SLF4j API.

When a test class is annotated as @UsingMocksAndStubs(Slf4jMocks.class), all production code touched by the tests in that class will receive mock Logger instances instead of real ones, when one of the factory methods in class org.slf4j.LoggerFactory is called.
