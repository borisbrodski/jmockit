Class Initializer

public final class Initializer
extends Object
implements org.testng.ITestListener

A test listener implementation for TestNG that will properly initialize JMockit before any tests are executed. This class is useful if running on JDK 1.6+ and the JVM argument "-javaagent:jmockit.jar" isn't being used.

One way to configure TestNG to use this class as a listener is to pass "-listener mockit.integration.testng.Initializer" as a command line argument. Another way would be through testng.xml configuration. Please check the TestNG documentation for more details.
