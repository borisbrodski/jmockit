JMockit: a Java toolkit for developer testing.

The figure below gives an overview of the testing APIs available in the toolkit. Clicking on individual items will navigate to the corresponding API documentation.


Figure 1. Overview of JMockit APIs

For a quick introduction to JMockit go to this page: Getting started. For the motivations that led to the creation of the toolkit and a brief introduction (with examples) to the main APIs, check out the About JMockit page.

The tutorial contains an extensive and user-friendly description of all the available tools and APIs, with plenty of example code.

The full distribution (in a zip file) contains several sample test suites written with the toolkit APIs. There are individual test suites for comparison with other popular mocking toolkits, providing the JMockit versions of the official example tests found on the Web for each other toolkit.
